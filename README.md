## Version

- node.js: 14.16.1 [다운로드](https://nodejs.org/ja/blog/release/v14.16.1/)
- npm: 6.14.12
```
npm i npm@6.14.12
```
- react: 17.0.2
- 이하 package.json 파일 확인.


## Running Process

### `install`
```
npm install
```

### `dev`

```
npm run dev
```

### `start`

```
npm run start
```

### `build`

배포 시에만 필요.(생략 가능)

```
npm run build
```
